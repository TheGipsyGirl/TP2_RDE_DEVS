# Trabajo Práctico - Portafolio Front End

Este proyecto es un **Trabajo Práctico (TP)** de Desarrollo Front End. Consiste en la creación de un portafolio personal utilizando HTML y CSS, con estructura semántica, diseño responsivo y publicación en GitHub Pages.
